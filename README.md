## General info
Wot Clans is web application for browsing ranking of best clans in World of Tanks

## Technologies
Project is created with:
* HTML
* CSS
* React.js (REST API)


	
## Setup
To run this project, install it locally using npm:

```
$ cd wot-clans
$ npm install
$ npm start
```
